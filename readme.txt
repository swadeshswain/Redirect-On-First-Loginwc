== Description ==


Suppose you have some information you want to redirect first time users to, such as a rules page, or a list of certain posts, etc. You would like to do that the first time they login as a new user.  You can do with using this  "Redirect ON First Login" Plugin.  In this Plugin you can see how to redirect the user to the change password page if it is the first time they have logged in.




= Redirect On First Login Features =

* Plugin config page
* User can active or deactive redirect funtion from plugin config page



HOW TO USE IT

* active the plugin option to "Yes" on plugin config page
* Add redirect URl on config page
